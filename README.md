# Black Box Eatz Website

This is a custom multi-page restaurant website for Black Box Eatz, a Black-owned Caribbean and Southern soul food fusion restaurant in Brooklyn, New York.

## Pages
- `index.html` - homepage
- `menu.html` - sample menu
- `gallery.html` - gallery page
- `about.html` - restaurant story and design choices
- `contact.html` - contact page
- `style.css` - all styling

## GitHub Pages Instructions
1. Create a new GitHub repository.
2. Upload every file from this zip directly into the repository. Do not upload the folder itself.
3. Go to Settings → Pages.
4. Under Branch, choose `main` and `/root`.
5. Click Save.
6. Open the GitHub Pages link after it publishes.
